Front-End Rendering

  • 图片：裁剪，模糊，亮度，对比度，阴影，灰度，色相旋转，反转，透明度，饱和度，褐色
    https://www.runoob.com/cssref/css3-pr-filter.html

  • 文字：模糊，阴影，透明度，发光，渐变

    https://codepen.io/wuyeguangming/pen/jONaWXr

  • 动效：位移，缩放，旋转，渐变
